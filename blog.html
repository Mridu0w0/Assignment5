Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select elements in a DOM using selectors. getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), or querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets the HTML content inside an element, including HTML tags.
innerText: Gets or sets the visible text inside an element, ignoring hidden elements and maintaining styling.
textContent: Gets or sets all text inside an element, including hidden elements, without interpreting HTML.

Question-3: What is event delegation in the DOM?

Event delegation is an event listener to a parent element instead of many child elements, so it can handle events as they bubble up.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and moves up through its parent elements in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element: let div = document.createElement("div")
Add to DOM: parent.appendChild(div)
Remove an element: parent.removeChild(child)
